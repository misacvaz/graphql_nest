# Dev

1. Clonar el Proyecto
2. Copiar el ```env.template``` y renmber a ```.env```

3. Ejecutar
```
yarn install

```
4. Levantar la imagem (Docker desktop)
```
docker-compose up -d

```

5. Levantar el backend de Nest
```
yarn start:dev

```
6. Visitar el sitio
```
localhost:3000/graphgl

```
